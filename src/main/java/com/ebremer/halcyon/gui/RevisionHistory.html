Version 0.0.0

03/07/2023
  • Text-based (RDF Turtle) annotations files using W3 Open Annotations (OA)
  • RDF data validation using SHACL
  • Separate OA compiler (ingest) for OA to Research Object Crate (ROC) files
  • POC WebGL Zephyr Viewer
  • Image and Feature Tiling Engines access based on IIIF
  • Image Tiling Engine currently supports TIFF, SVS, NDPI
  • Image libraries based on OME Bioformats
  • Identity management via embedded Keycloak
  • Security Model based on Web Access Control
  • Data accessible via SPARQL Endpoint
  • Viewer code based on OpenSeadragon
  • Server UX built with Apache Wicket
  • Main data storage based on Apache Jena TDB2
  • Java environment based on GraalVM 22.3.1 (JDK 17)
  • Semantic versioning
